📖 为什么写成 ⟨ψ|σz|ψ⟩？（点击展开）

⟨ψ| 是什么？

⟨ψ| 是 |ψ⟩ 的共轭转置（也叫对偶矢量）：
|ψ⟩ = α|↑⟩ + β|↓⟩ 是列向量
⟨ψ| = α*⟨↑| + β*⟨↓| 是行向量（*表示复数共轭）

第1步：σz 作用在 |ψ⟩ 上

σz 作用在本征态上：
σz|↑⟩ = +1|↑⟩，σz|↓⟩ = -1|↓⟩

所以 σz|ψ⟩ = σz(α|↑⟩ + β|↓⟩) = α|↑⟩ - β|↓⟩

第2步：与 ⟨ψ| 做内积

内积 ⟨ψ|(σz|ψ⟩) 展开：
[α*⟨↑| + β*⟨↓|] × [α|↑⟩ - β|↓⟩]

= α*α⟨↑|↑⟩ + α*(-β)⟨↑|↓⟩ + β*α⟨↓|↑⟩ + β*(-β)⟨↓|↓⟩

📐 基矢的矩阵表示（点击展开）

我们约定基矢的矩阵形式：

|↑⟩ = [1, 0]ᵀ = 列向量 (朝上态)
|↓⟩ = [0, 1]ᵀ = 列向量 (朝下态)

对应的左矢（行向量）：

⟨↑| = [1, 0] = 行向量
⟨↓| = [0, 1] = 行向量

内积计算验证：

⟨↑|↓⟩ = [1, 0] × [0, 1]ᵀ = 1×0 + 0×1 = 0
⟨↑|↑⟩ = [1, 0] × [1, 0]ᵀ = 1×1 + 0×0 = 1
⟨↓|↓⟩ = [0, 1] × [0, 1]ᵀ = 0×0 + 1×1 = 1

✓ 这就是正交归一性！

第3步：正交归一性简化

因为 ⟨↑|↑⟩=1，⟨↓|↓⟩=1，⟨↑|↓⟩=⟨↓|↑⟩=0
所以结果 = α*α - β*β = |α|² - |β|²

✅ 这就是期望值 ⟨σz⟩，等于多次测量的平均值！

💡 通俗解释本征值与期望值（点击展开）

🎯 什么是本征值？

本征值就是"标准答案"！当算符 σz 作用在 |0⟩（朝上）时，结果是 +1×|0⟩，所以 +1 是 |0⟩ 的本征值；作用在 |1⟩（朝下）时，结果是 -1×|1⟩，所以 -1 是 |1⟩ 的本征值。

🧪 本征态就像"训练有素的学生"，每次考试都得同一个分数！

📊 期望值公式怎么理解？

公式 ⟨σz⟩ = Σ λᵢ P(λᵢ) 意思是：

期望值 = (各个可能结果) × (对应概率) 的总和

对于 σz：

⟨σz⟩ = (+1) × P(+1) + (-1) × P(-1)
= (+1) × |α|² + (-1) × |β|²
= |α|² - |β|²

✅ 这就是多次测量的平均值！

🔬 泡利算符 σz 是什么？

简单说：算符就是一道"测量题"，σz 就是"测一下自旋朝上还是朝下"这道题目。

💡 类比：就像老师出一张试卷（算符），你做这道题得到分数（测量结果）。这份试卷有两个固定的"标准答案"：+1分和-1分。

两个固定结果：

  • 如果粒子是 |0⟩（朝上）→ 测量结果永远是 +1
  • 如果粒子是 |1⟩（朝下）→ 测量结果永远是 -1

所以 |0⟩ 和 |1⟩ 是 σz 的"标准答案态"，+1 和 -1 就是标准答案。

叠加态测出来是什么？

粒子处于叠加态 |ψ⟩ = α|0⟩ + β|1⟩ 时，测量结果随机：有 |α|² 的概率得 +1，|β|² 的概率得 -1。

平均分 = (+1)×考到+1的概率 + (-1)×考到-1的概率
⟨σz⟩ = (+1)×|α|² + (-1)×|β|² = |α|² - |β|²
这就是"期望值"，也叫平均分！

📐 矩阵运算怎么算？

物理学家用矩阵乘法来算上面的"平均分"，流程很简单：

第1步：把态矢量写成竖着的清单：

|ψ⟩ = [α, β]ᵀ   （α和β是两份"配料"）
⟨ψ| = [α, β]   （横向展开，方便计算）

第2步：算符是一张2×2的"价目表"：

σz = [[1, 0], [0, -1]]   （+1和-1分别是两个结果的分数）

第3步：三样东西按顺序相乘：

⟨σz⟩ = [α, β] × [[1,0],[0,-1]] × [α, β]ᵀ
= [α, β] × [α, -β]ᵀ
= α² - β²
📐 详细推导过程（点击展开）

第1步：明确符号

[α, β] 是行向量，
[α, β]ᵀ 是列向量，
中间的 × 是普通矩阵乘法。

所以表达式是：

[α   β] × [[1, 0],[0, -1]] × [α, β]ᵀ

第2步：行向量 × 矩阵

[α   β] × [[1, 0],[0, -1]]

= [α×1 + β×0,   α×0 + β×(-1)]

= [α,   -β]

第3步：再乘列向量

[α,   -β] × [α, β]ᵀ

= α×α + (-β)×β

= α² - β²
✅ 最终结果：α² - β²
💡 一句话总结：把α和β投入算符这套"机器"，出来的结果就是 α² - β²，也就是测到正面的概率减去测到反面的概率。
